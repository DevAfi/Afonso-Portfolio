About This Project

The Secure Password Manager CLI is a comprehensive, local-first password management solution built with military-grade encryption standards. This project demonstrates advanced cryptographic principles and secure software development practices, showcasing expertise in Python security applications.

Key Features

  • Master Password Authentication: Secure login system with attempt limiting
  • PBKDF2 Key Derivation: 100,000 iterations with salt for secure key generation
  • Fernet Encryption: AES-128 + HMAC for authenticated encryption
  • Local JSON Storage: Encrypted data persistence with metadata
  • Interactive CLI Menu: User-friendly command-line interface
  • Secure Password Input: Hidden character input using getpass
  • Attempt Limiting: Brute force protection with 4-attempt limit

Technical Implementation

üîê Security Architecture

The application implements a robust security model based on industry-standard cryptographic practices:

  • Key Derivation: PBKDF2 with SHA-256 and 100,000 iterations
  • Encryption: Fernet (AES-128 + HMAC) for authenticated encryption
  • Salt Generation: Cryptographically secure random salt for each session
  • Password Hashing: Master password never stored in plaintext

Getting Started

Installation

# Clone the repository
git clone https://github.com/DevAfi/Encryption-Password-Manager.git
cd Encryption-Password-Manager

# Install dependencies
pip install cryptography pyperclip

# Run the application
python main.py

First Time Setup

The application automatically detects if no master password exists and will prompt you to create one. The master password must be at least 8 characters long and is securely hashed using PBKDF2.

Main Features

  • Add Password: Store encrypted credentials for any service
  • Retrieve Password: View stored passwords by selecting from a list
  • Secure Storage: All data encrypted at rest using Fernet
  • Local Only: No cloud dependencies, complete privacy
