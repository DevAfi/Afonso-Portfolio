About This Project

FinancePal is a sophisticated, feature-rich personal finance management application that demonstrates advanced frontend development skills with comprehensive analytics and dynamic visualizations. This portfolio-ready project showcases expertise in vanilla JavaScript, data visualization, and modern responsive design principles.

Key Features

  • Smart Transaction Entry: Add income/expenses with categories, dates, and automatic categorization
  • Persistent Local Storage: All data saved locally with automatic backup and recovery
  • Real-time Updates: Instant balance calculations and visual updates across all components
  • Data Export: Export transactions to CSV and JSON formats for external analysis
  • Advanced Analytics Dashboard: Comprehensive financial insights with interactive charts
  • Financial Health Score: AI-powered scoring system (0-100 scale) with detailed metrics
  • Smart Budget Management: Set category budgets with progress tracking and intelligent alerts
  • Premium User Experience: Dynamic backgrounds, custom cursor effects, and glassmorphism design
  • Fully Responsive: Optimized for all devices with mobile-first approach

Technical Implementation

üéØ Financial Health Scoring Algorithm

The sophisticated scoring system evaluates financial health across multiple dimensions:

  • Savings Rate (40 points): Income vs expenses ratio calculation
  • Expense Stability (30 points): Consistency analysis of monthly spending patterns
  • Category Balance (30 points): Diversification assessment of expense categories
  • Health Metrics: Excellent (80-100), Good (60-79), Fair (40-59), Poor (0-39)
